Database Systems Laboratory - Key Learnings
This folder contains the practical implementations and laboratory exercises performed throughout the course. The following core SQL concepts were explored and mastered:

1. 🛠️ Environment & Tools
Oracle Database XE: Hands-on experience with the Oracle Express Edition environment.
SQLPlus Tool: Proficient in using the command-line interface (SQLPlus) for executing scripts and managing database sessions.

2. 🏗️ Data Definition & Manipulation (DDL & DML)
Intermediate DDL: Experienced in creating, altering, and dropping complex table structures, including constraints (Primary Key, Foreign Key, Unique, Not Null, and Check).
Simple DML: Practical knowledge of inserting, updating, and deleting records while maintaining data integrity.

3. 🔍 Advanced Querying Techniques
Multi-table Joins: Retrieving data from multiple related tables using INNER JOIN, LEFT/RIGHT OUTER JOIN, and FULL OUTER JOIN.
Set Operations: Implementing UNION, INTERSECT, and MINUS to combine results from different queries.
Aggregate Functions: Summarizing data using COUNT(), SUM(), AVG(), MAX(), and MIN() along with the GROUP BY and HAVING clauses.

4. 🧩 Complex Logic & Subqueries
Nested Subqueries: Writing queries within queries to solve multi-level data retrieval problems.
Complex Joins: Mastering the use of Outer Joins to handle null values and unmatched records across tables.
